# EEE-_443_Speech_Emotion_Recognition

## Emotional Speech Classification

This project explores speech emotion recognition using deep learning models.
The CREMA-D dataset is used with MFCC feature extraction.

### Models
- BiLSTM
- Transformer
- 1D ResNet

### Report
A detailed experimental report including methodology, hyperparameter tuning,
and evaluation results is available in pdf.

### How to Run
python main.py
